# 🌱☀ 영농형 태양광 관리 프로그램

농축산(가축·작물)과 태양광 발전을 **한 화면에서 통합 관리**하는 프로그램입니다.
두 가지 방식으로 쓸 수 있습니다.

| 방식 | 어디서 | 추천 상황 |
|------|--------|-----------|
| 🌐 **웹 버전** (`web_app.py`) | PC 브라우저 + **휴대폰**(같은 와이파이/홈화면 설치) | 휴대폰에서도 쓰고 싶을 때 — **권장** |
| 🖥 **데스크톱 버전** (`app.py`) | macOS / Windows 창 프로그램 | PC 한 대에서만 쓸 때 |

데이터는 두 방식 모두 **같은 SQLite 파일(`farm_data.db`)** 을 씁니다.

---

## 🌐 웹 버전 (PC + 휴대폰) — 권장

```bash
cd farm_solar_manager
python3 -m pip install -r requirements.txt   # 최초 1회 (Flask 설치)
python3 web_app.py
```

- 또는 더블클릭 실행: **macOS** `run_macos.command` · **Windows** `run_windows.bat`
  (Flask가 없으면 자동 설치 후 브라우저를 띄웁니다.)
- 이 PC에서 접속: **http://localhost:8000**

### 📱 휴대폰에서 쓰기
1. PC와 휴대폰을 **같은 와이파이**에 연결합니다.
2. 서버를 켜면 콘솔에 `http://192.168.x.x:8000` 형태의 **휴대폰 접속 주소**가 표시됩니다.
3. 휴대폰 브라우저에 그 주소를 입력해 접속합니다.
4. **홈 화면에 추가**하면 앱 아이콘처럼 쓸 수 있습니다(PWA).
   - 아이폰(Safari): 공유 → "홈 화면에 추가"
   - 안드로이드(Chrome): 메뉴 → "앱 설치 / 홈 화면에 추가"

> 휴대폰이 접속하는 동안에는 **PC(서버)가 켜져 있어야** 합니다.

### ☁ PC를 꺼도 어디서나 접속 (외부망 배포)
PC를 켜두지 않아도 인터넷 어디서나 쓰려면 **무료 클라우드(PythonAnywhere)** 에 올리면 됩니다.
단계별 안내: **[DEPLOY.md](DEPLOY.md)** 참고 (약 15분, 비기술자용으로 작성).
배포본은 **비밀번호 보호**(환경변수 `APP_PASSWORD`)가 켜져 아는 사람만 접속할 수 있습니다.

### 📄 서류 바로가기
앱의 **📄 서류** 탭에서 토지대장·건축물대장·등기부등본, 태양광 SMP/REC·계통연계·사용전검사,
가축사육업·가축분뇨 신고, 농업경영체 등록 등 **자주 쓰는 정부 민원 사이트로 바로 이동**합니다.
관할 시·군·구청이나 조합 사이트를 추가하고 싶으면 `web_app.py` 의 `LINK_GROUPS` 목록만 고치면 됩니다.

## 🖥 데스크톱 버전 실행

```bash
cd farm_solar_manager
python3 app.py
```

> Python 3.8 이상이면 됩니다(Tkinter 포함). 처음 실행하면 데이터 파일 `farm_data.db` 가 생성됩니다.

> **⚠ 창이 검게만 보이나요?** macOS 기본 파이썬은 오래된 **Tk 8.5** 를 써서 최신 macOS에서
> 창이 검게 뜨는 알려진 버그가 있습니다. **Tk 8.6 이상**이 있는 파이썬으로 실행하면 해결됩니다:
> `brew install python@3.12 python-tk@3.12` 후 `/opt/homebrew/bin/python3.12 app.py`,
> 또는 [python.org](https://www.python.org/downloads/) 설치본 사용(Tk 8.6 기본 포함).
> 빌드 스크립트(`build_macos.sh`)는 Tk 8.6 파이썬을 **자동으로 찾아** 빌드합니다.

**처음이라면** 상단 메뉴 `파일 > 예시 데이터 채우기` 를 눌러
화면이 어떻게 보이는지 미리 확인해 보세요. (`파일 > 전체 데이터 초기화` 로 지울 수 있습니다.)

## 주요 기능 (5개 탭)

| 탭 | 내용 |
|----|------|
| 🏠 **대시보드** | 진행중 품목, 이번 달 발전량·발전수익, 올해 순이익을 카드로 요약 + 최근 발전량 추이 |
| 🐄 **농축산** | 사육/재배 품목 관리 + 영농 일지(급여·방역·수확·출하) 기록 |
| ☀ **태양광** | 발전 설비 단가 설정 + 일별 발전량(kWh) 입력 → **예상 수익 자동 계산** |
| 💰 **재무** | 수입/지출을 농축산·태양광·공통 부문별로 정리, 총수입·총지출·순이익 표시 |
| 📊 **통계** | 연도별 월간 수입/지출 막대그래프 + 월별 발전량 추이 꺾은선그래프 |

## 태양광 수익 계산 방식

`[태양광]` 탭에서 설정한 단가로 발전 수익을 자동 계산합니다.

```
1kWh당 정산단가 = SMP 단가 + (REC 단가 × REC 가중치)
예상 수익        = 발전량(kWh) × 1kWh당 정산단가
```

기본값(예시): SMP 130원 + REC 70원 × 가중치 1.2 = **214원/kWh**
→ 실제 계약 단가에 맞게 `[태양광] > 발전 설비 설정` 에서 수정하세요.

## 데이터 저장 / 백업

- 모든 데이터는 **`farm_data.db` 파일 하나**(SQLite)에 저장됩니다.
- **백업**: 이 파일을 복사해 두면 됩니다. (다른 PC로 옮길 때도 파일만 복사)
- 위치 확인: 메뉴 `도움말 > 정보` 에 전체 경로가 표시됩니다.

## 엑셀로 내보내기

메뉴 `파일 > CSV로 내보내기…` 를 누르고 저장할 폴더를 고르면 5개의 CSV가 만들어집니다.

- `농축산_품목.csv` · `영농_일지.csv` · `태양광_발전기록.csv` · `재무_거래내역.csv`
- `월별_요약.csv` — 연·월별 수입/지출/순이익/발전량/발전수익을 한 표로

> 한글이 깨지지 않도록 `utf-8-sig`(BOM)로 저장되어 **엑셀에서 더블클릭하면 바로** 열립니다.

---

## 발전수익 → 재무 자동 반영

`[태양광]` 탭의 **`💰 발전수익 재무 반영`** 버튼을 누르면, 월별 발전량 × 단가로
계산한 예상 수익이 `[재무]` 탭에 **`발전 정산금(자동)`** 수입 항목으로 들어갑니다.

- 같은 달은 **중복 없이 갱신**됩니다(여러 번 눌러도 안전).
- 단가를 바꾼 뒤 다시 누르면 금액이 새로 계산됩니다.

## 실행 파일로 만들기 (파이썬 없는 PC에 배포)

파이썬이 깔리지 않은 컴퓨터에서도 **더블클릭으로 실행**되는 파일을 만들 수 있습니다.

**macOS:**
```bash
bash build_macos.sh        # → dist/FarmSolarManager.app
```

**Windows:** (Windows PC에서) `build_windows.bat` 더블클릭
```
                           → dist\FarmSolarManager.exe
```

> - 빌드에는 PyInstaller가 필요하며 스크립트가 자동으로 설치합니다.
> - `.app`은 macOS에서, `.exe`는 Windows에서 각각 만들어야 합니다(교차 빌드 불가).
> - **실행 파일로 배포하면** 데이터는 `문서/영농형태양광관리/farm_data.db` 에 저장됩니다.

## 자주 묻는 점

- **날짜 입력 형식**: `YYYY-MM-DD` (예: 2026-06-15). 칸에 오늘 날짜가 기본으로 채워집니다.
- **같은 날 발전량을 또 입력하면?** 덮어쓰기 됩니다(하루 1건 원칙).
- **품목을 삭제하면 일지는?** 일지는 남고 품목 표시만 '미지정'으로 바뀝니다.
